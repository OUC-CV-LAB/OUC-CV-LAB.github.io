参与学术会议

第五届多媒体与人工智能安全学术研讨会
时间：2026年4月24日-26日
地点：安徽省黄山市香茗国际酒店
主办方：安徽省人工智能学会、隐者联盟
合肥工业大学承办并参与第五届多媒体与人工智能安全学术研讨会。会议面向国际学术前沿与国家战略需求， 聚焦多媒体安全、人工智能安全、数据安全等前沿技术与热点领域，邀请50 余位行业权威专家做特邀报告， 设置数据安全、多媒体内容安全、人工智能安全、女科学家论坛等多个特色论坛。这场高水平学术盛宴汇聚了 来自学术界与产业界的专家学者、青年才俊，各论坛报告内容丰富、精彩纷呈，全面展示多媒体与人工智能安 全领域最新研究成果与行业发展动态。与会专家积极交流探讨、思想碰撞充分，学术研讨氛围浓厚。
人工智能前沿论坛・第四期
时间：2024年12月26日
地点：中国科学技术大学高新校区
主办方：脑启发智能感知与认知教育部重点实验室
由脑启发智能感知与认知教育部重点实验室主办，华为诺亚方舟实验室、类脑智能技术及应用国家工程实验室和安徽省计算机学会机器学习专委会协办的第四期人工智能前沿论坛暨“具身智能”专题论坛在中国科学技术大学高新校区成功举办。 本次论坛邀请了19位国内人工智能领域知名专家，围绕“具身智能”专题进行了精彩的学术报告。吸引了来自中国科学技术大学、复旦大学、山东大学、山东师范大学、中国海洋大学、中科院合肥创新院等高校和科研院所的师生及专家学者们，线下总人数达到300余人。
CSIG青年科学家会议
时间：2024年12月13日
地点：杭州
主办方：中国图象图形学学会
大会特别邀请了四位在图像图形学领域取得卓越成就的专家做主旨报告。他们分别是南京大学副校长、国家杰出青年科学基金获得者周志华教授，中国科学院自动化研究所多模态人工智能系统全国重点实验室副主任王亮研究员，香港科技大学计算机科学与工程学系教授、思谋科技创始人贾佳亚教授和深圳大学计算机与软件学院院长黄惠教授。四位讲者就各自领域的最新研究成果发表了精彩的报告。 第二十一届中国图象图形学学会青年科学家会议拟于2025年在青岛召开。成功申办2025年CSIG科学家会议，并入选青工委。

举办学术活动

合肥工业大学斛兵学术前沿论坛（第二期）
时间：2026年4月27日-28日
地点：合肥工业大学(屯溪路校区)学术会议中心二楼小报告厅
主办方：合肥工业大学计算机与信息学院,人工智能创新学院
合肥工业大学计算机与信息学院、人工智能创新学院联合主办斛兵学术前沿论坛（第二期）。 会议面向国际学术前沿与国家战略需求，聚焦脑机接口、脑科学、神经工程、医学工程、类脑计算 等前沿交叉领域，汇聚来自浙江大学、华中科技大学、哈尔滨工业大学、中国科学院深圳先进技术研究院、 华南理工大学、天津大学、杭州电子科技大学等国内顶尖高校与科研院所的脑机接口领域权威专家。汇聚 人工智能、脑科学、神经工程、医学工程、类脑计算等领域顶尖学者，报告内容覆盖脑电信号精准解码、 无创脑机接口关键技术、类脑计算等前沿方向，全方位展示最新科研成果、技术突破与产学研协同发展路径。
合肥工业大学斛兵学术前沿论坛（第一期）
时间：2026年4月18日-19日
地点：合肥工业大学(翡翠湖校区)科教楼B座501报告厅
主办方：合肥工业大学计算机与信息学院,人工智能创新学院
合肥工业大学计算机与信息学院、人工智能创新学院联合主办斛兵学术前沿论坛（第一期）。 会议面向国际学术前沿与国家战略需求，聚焦人工智能、高性能计算、生命健康、医疗影像、 空基无人系统、类脑智能等前沿技术与热点领域， 汇聚来自中山大学、上海交通大学、湖南大学、上海大学、浙江大学、电子科技大学、 中国科学院等多所高校与科研院所的国家级高层次人才。汇聚计算机、 人工智能、医学影像、机器人、感知计算等领域的顶尖专家，报告内容覆盖超算互联网、 具身智能、大模型智能体、多模态健康感知、肿瘤诊疗、AI 成像、手术导航、类脑计算等前沿方向， 全方位展示最新科研成果与产业应用进展。
CSIG2025青年科学家会议
时间：2025年9月20日
地点：山东省青岛市西海岸新区银沙滩温德姆至尊酒店
主办方：中国图象图形学学会
中国海洋大学蔡青副教授承办并参与CSIG2025。会议面向国际学术前沿与国家战略需求，聚焦最新前沿技术和热点领域，精彩呈现了4个大会特邀报告， 30场主题分论坛，4场讲习班，4场学术服务论坛，8场青年学者服务论坛，总计200余场高水平学术报告。这场精彩的学术盛宴吸引了超过 2000位来自学术界和产业界的专家学者、青年才俊，超过350家单位共同参与。大会分论坛200余场学术报告精彩纷呈，为与会者们呈现了最前沿的科技成果和行业动态。 资深学者进行评点，与会者们积极参与，讨论氛围热烈。
